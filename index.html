رجکس تشخیـص شمـاره تلفن ایرانـی

صحت سنجی شماره تلفن ایرانی با رجکس

مجموعۀ RegEx های بهینۀ تشخیص شماره تلفن ایرانی، برای همۀ توسعه دهندگان ایرانی

چرا رجکس؟

ممکن است برای شما سوال باشد چرا باید از رجکس استفاده کنیم وقتی میتوانیم از if ها و else ها استفاده کنیم و این‌همه عبارت رو یاد نگیریم؟ باید به شما بگوییم که توالی های تودرتو شرطی، فقط کد های شما رو کثیف تر میکنه و بعد ها اگر ایرادی در شرط های شما باید، خطایابی از آن کد ها خیلی سخت تر صورت میگیره. اینجا چند دلیل داریم برای استفاده از رجکس ها :

سریع و بهینه

سریع و بهینه در همه نرم‌افزار ها.

ساده و قابل فهم

ساختاری ساده و قابل درک برای همه.

کاربردی

کاربرد های متنوع و عالی در همه‌جا.

روش درست

روش اصولی بررسی string ها.

کد تمیز تر

رجکس ها، تمیز تر از if و else ها
